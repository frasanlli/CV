pdf_to_csv

Página web en la que cargar archivos pdf y extraer datos en formato .csv

  • Uso JS y la librería pdf.js
  • Importar pdf para marcar áreas en tablas complejas.
  • Obtener plantilla de áreas para reutilizar y csv con texto marcado
  • Versión para pantallas táctiles pendiente de algunas mejoras

calibration-mail

Aplicación web para gestionar herramientas en un servidor local.

  • Uso de Django y base de datos sqlite3 para gestionar uso y necesidad de calibración de dispositivos de medida.
  • Posibilidad de importar xlms para transferirlo a la base de datos.
  • Correos automatizados con estado de los elementos.

supermarket_prices

Este programa permite obtener los precios y otra información relevante sobre los productos extrayéndolos de los sitios web de los supermercados.

  • Python con Selenium y Pandas para obtener productos de diversos supermercados y filtrarlos
  • Uso JSON y Firebase (noSQL online) para almacenar los datos extraídos
  • Creación de log para registrar ejecución y errores
  • Inicio de sesión y roles de usuario
